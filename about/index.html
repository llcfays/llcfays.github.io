2016/02-2017/2 上汽e家

  • 软件环境：Android 4.0至Android 7.1.2
  • 开发环境：Android Studio+Git
  • 项目描述：上汽集团党员群众 App。
  • 技术要点：
    • 1、应用以自定义 View +懒加载 ViewPager 为框架。
    • 2、通过 OkHttp+Retrofit 访问网络，对 Retrofit 进行封装实现
      Session 过期自动登录和使用 Gson 进行 Json 解析。
    • 3、使用 Glide 进行图片加载，对 Glide 进行封装，对内存占用进行
      大幅度的优化，防止 ANR 和 OOM。
    • 4、使用极光推送、GreenDao 和自定义View实现IM功能。
    • 5、优化 Adapter 为 RecyclerView 进行大量数据和多图片的加载。
    • 6、使用 RxJava 方便开发。
    • 7、差异化版本构建。
  • 项目职责：
    • 1、担任 Android App 开发人，负责搭建项目整体架构。
    • 2、负责 App 应用模块界面及逻辑实现。
    • 3、项目代码优化和后期功能更改、代码维护，如优化排查解决内存
      泄漏和应用对内存的占用；排查解决界面卡顿问题，优化应用对
      CPU和GPU的占用，大幅降低应用卡顿几率。
    • 4、负责解决客户端兼容适配问题，如不同手机调用相机闪退、
      Android M 的 Runtime Permission 适配问题等。
    • 5、负责客户端代码重构工作。包括对项目主结构进行重新划分和抽
      象，对引入的开源框架进行抽象封装，尽量是整个项目呈现高内
      聚，低耦合。
